Module_a 문제해결방안

- 데이터 읽기

연도별 CSV 파일 경로를 사전에 정의하고 pandas를 사용하여 데이터 로드했고 파일 인코딩 문제(CP949 등)로 인해 오류 발생 가능성을 try-except로 처리하였다.

- 데이터 가공

이전 버전과 마찬가지로 유형 컬럼을 기준으로 데이터를 그룹화했다. 각 유형별로 "남자"와 "여자" 데이터를 키-값 쌍으로 정리했다. 

- JSON 파일 저장

가공한 데이터를 연도별 JSON 파일로 저장하였고 저장 과정 중 발생 가능한 파일 접근 문제를 try-except로 처리했다.

- 카테고리 리스트 생성 및 예외처리

특정 연도에 해당하는 JSON 파일을 읽어와 키값을 추출하여 키값에 번호를 붙여 카테고리 리스트로 반환하였다.
이때 JSON 파일이 없는 경우 오류메시지를 출력하도록 하였다.
